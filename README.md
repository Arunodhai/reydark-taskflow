# ReyDark TaskFlow (RD-TaskFlow)

TaskFlow is a lightweight task management SaaS platform developed by **ReyDark**.  
The goal is to build a scalable, real-world system that enables teams to create, assign, and track tasks efficiently.

---

## 🚀 Project Overview

TaskFlow is being developed as part of a simulated company environment to replicate real-world software engineering workflows, including:

- Agile sprint cycles
- Jira-based task management
- GitHub pull request workflows
- Slack-based team communication
- Structured documentation and decision tracking

---

## 🔄 Development Workflow

This project follows a structured workflow similar to real-world teams:

1. Tasks are created and managed in **Jira**
2. Work is tracked using sprint cycles
3. Code is developed in feature branches
4. Pull Requests (PRs) are created for review
5. Changes are merged after validation

---

## 📋 Issue Tracking

All tasks, bugs, and features are tracked in **Jira**.

- Epics → High-level features  
- Stories → User-facing functionality  
- Tasks → Technical implementation  
- Bugs → Defects and fixes
